# Changelog

This project ships source-only Chrome extension updates. No release bundles or build artifacts are published.

## 1.0.1 - 2026-07-06

### Changed

- Include `file://` tabs in tab bankruptcy counts, bookmarks, and optional tab closing.
- Treat URL scheme filtering case-insensitively.
- Document that Chrome's per-extension **Allow access to file URLs** toggle is required for local file tab support.
- Bump the extension manifest version from `1.0.0` to `1.0.1`.
